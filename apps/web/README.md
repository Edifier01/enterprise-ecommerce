# Next.js Storefront

App Router + TypeScript. Initialize shadcn: `npx shadcn@latest init`

## Run

```bash
npm install
npm run dev
```

Open http://localhost:3000

## Structure

```
src/
├── app/           # routes
├── components/    # ui/ for shadcn
├── features/      # catalog, cart, checkout
├── lib/           # api client
└── store/         # Zustand
```

See `.cursor/rules/frontend/`.
